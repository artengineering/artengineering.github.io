FPGA BASED CONTROLLER

Designed to work as a part of the modular backplane based system for protecting superconducting magnets (Digital Quench Detection System). Controller interacts with other modules via backplane.

What I have done:

  • Generated idea
  • Contributed to generating functional and technical specification
  • Designed complete schematic in Altium
  • Chose components for the board
  • Ordered components
  • Did the layout of the PCB in Altium
  • Ordered the PCB and PCB assembly
  • Prepared detailed test plan
  • Troubleshooted and tested the board
  • Wrote HDL code for FPGA for testing PCB
  • Wrote HDL code for FPGA for production
  • Wrote test benches' HDL code for simulation in Vivado Simulator
  • Tested and debugged Verilog modules in Vivado Simulator
  • Performed integration test of the completed Verilog project
  • Test benches use text file as a source of stimulus and automaticaly generate verification report
  • Generated the TCL script for recreating the project in Vivado from source files
  • Designed the front panel of the controller in AutoCAD

PCB features and characteristics:

  • DC-DC With 4 outputs: 3.3V, 1.8V, 1.5V, 1.0V
  • Xilinx Z7C020
  • DDR3 SDRAM memory 256 MB
  • Ethernet 100Mb/1Gb
  • Flash memory
  • EEPROM memory
  • 8 optoisolated digital outputs
  • 8 optoisolated digital inputs
  • Linear regulator +11v, -11V for analog part
  • SAR ADC 24 bit
  • SAR ADC 16 bit
  • DAC 18 bit

Verilog project:

  • SPI controller for communication with GUI via NI cRIO
  • Communication protocol
  • SPI multiplexer for module communication via backplane
  • SPI driver for ADC
  • Register file with memory access through SPI instructions
  • Circular buffer for ADC samples
  • Data processing and FSM based protection logic
