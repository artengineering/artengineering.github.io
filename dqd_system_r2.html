MAGNET PROTECTION SYSTEM

Designed to add new features to the existing system and simplifies the assembling process.

What I have done:

  • Generated the idea of developing completely new system for protecting superconducting magnets
  • Pitched the idea to the managment and got funding
  • Contributed to generating functional and technical specification
  • Designed blog-diagram of the whole system
  • Chose components for the chassis
  • Ordered components
  • Prepared detailed written test plan
  • Troubleshooted and tested the integrated system

System features and characteristics:

  • Up to 8 modules per one chassis
  • Up to 32 (4 per module) individually configurable (gain, attenuation, filter) 16b ADC channels with 1.5kV isolation
  • 24b ADC channel for measuring power supply current
  • Up to 8 optoisolated digital outputs to interface with other systems
  • Up to 8 optoisolated digital inputs to interface with other systems
  • SPI interface with host PC through the NI cRIO
  • Capability of detecting unconnected signal wires
  • Each module has local SRAM based circular buffer capable of storing 10ch x 60ksamples
  • Front panel indication of the most important signals
