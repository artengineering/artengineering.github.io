FPGA BASED DQD MODULE

Designed to work as a part of the modular backplane based system for protecting superconducting magnets (digital quench detection system).

What I have done:

  • Generated idea
  • Contributed to generating functional and technical specification
  • Designed complete schematic in Altium
  • Chose components for the board
  • Ordered components
  • Did the layout of the PCB in Altium
  • Ordered the PCB and PCB assembly
  • Prepared detailed test plan
  • Troubleshooted and tested the board
  • Wrote HDL code for FPGA for testing PCB
  • Wrote HDL code for FPGA for production
  • Wrote test benches' HDL code for simulation in Vivado Simulator
  • Tested and debugged Verilog modules in Vivado Simulator
  • Performed integration test of the completed Verilog project
  • Test benches use text file as a source of stimulus and automaticaly generate verification report
  • Generated the TCL script for recreating the project in Vivado from source files
  • Designed the front panel of the controller in AutoCAD

PCB features and characteristics:

  • 4 isolated (1500V) low-noise 16b ADC channels:
  • Signal conditioning with settable gain or attenuation and settable lowpass filter
  • Isolated SPI interface between analog and digital part
  • Isolated I2C interface between analog and digital part
  • DC-DC with 3 outputs for digital part: 3.3V, 1.8V, 1.0V
  • Xilinx Artix-7 FPGA XC7A100TCSG324
  • SRAM memory 1x16Mb
  • Flash memory
  • EEPROM memory

Verilog project:

  • SPI controller for communicating with GUI via NI cRIO
  • Communication protocol
  • SPI driver for ADC
  • I2C driver for interfacing with digital expander
  • Register file with memory access through SPI instructions for system configuration and status
  • SRAM driver
  • SRAM based circular buffer for ADC samples
  • Data processing and protection logic (threshold crossing)
